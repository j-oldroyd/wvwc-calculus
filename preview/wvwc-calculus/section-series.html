    Section 9.2 Series

    Consider the number \(\pi = 3.14159\ldots\text{.}\) This number is irrational and so cannot be represented as a rational number \(\frac{a}{b}\text{.}\) This leads to the question of what we mean by \(\pi\text{?}\) Or in particular, how can we actually make sense of \(\pi\text{,}\) or represent it?
    🔗
    We can consider rewriting \(\pi\) as follows:
    \begin{equation*} \pi = 3.14159\ldots = 3 + \frac{1}{10} + \frac{4}{100} + \frac{1}{1000} + \frac{5}{10000} + \frac{9}{100000} + \cdots\text{.} \end{equation*}
    So we can identify \(\pi\) with the sequence \(\{a_{n}\}_{n=0}^{\infty} = \{3, \frac{1}{10}, \frac{4}{100}, \ldots\}\) and the series
    \begin{equation*} a_{0} + a_{1} + a_{2} + \cdots\text{.} \end{equation*}
    🔗

    Subsection Infinite Series

    We now introduce the fundamental construction of this chapter.
    🔗

    Definition 9.2.1. Infinite Series.

    An infinite series is a sum of the form
    \begin{equation*} \sum_{k=0}^{\infty}a_{k} = a_{0} + a_{1} + \cdots \end{equation*}
    where \(\{a_{k}\}_{k=0}^{\infty}\) is a sequence.
    🔗
    🔗
    Infinite series are useful for representing (and computing) irrational numbers (which includes almost all numbers).
    🔗

    Example 9.2.2. Guessing sums.

    Guess the sums of the following series:
    1. \(\displaystyle \sum_{k=0}^{\infty}2^{k}\)
      🔗
    2. \(\displaystyle \sum_{k=1}^{\infty}2^{-k}\)
      🔗
    3. \(\displaystyle 1 - 1 + 1 - 1 + 1 - \cdots\)
      🔗
    🔗
    Solution.
    We have the following:
    1. \(\displaystyle \sum_{k=0}^{\infty}2^{k} = \infty\)
      🔗
    2. \(\displaystyle \sum_{k=1}^{\infty}2^{-k} = 1\)
      🔗
    3. For this last sum we have an issue: there’s no sensible way to define this sum. We can say that \(1 - 1 + \cdots = 0\) or \(1\) by grouping terms differently.
      🔗
      🔗
    🔗
    🔗
    🔗
    We can determine what the value of a series should be by using limits.
    🔗

    Definition 9.2.3. Partial Sums and Convergence.

    Given the series \(\sum_{k=0}^{\infty}a_{k}\text{,}\) we denote its \(n^{\text{th}}\) partial sum by
    \begin{equation*} S_{n} = \sum_{k=0}^{n}a_{k}\text{.} \end{equation*}
    If the sequence \(\{S_{n}\}\) is convergent and \(S_{n}\to S\text{,}\) then we say that the original series converges and \(\sum_{k=0}^{\infty}a_{k} = S\text{.}\) If the sequence of partial sums diverges, we say the original series diverges.
    🔗
    🔗
    Using Definition 9.2.3, we can say that the sum \(1 - 1 + 1 - 1 + \cdots\) diverges, since its sequence of partial sums is \(\{1,0,1,0,1,\ldots\}\text{.}\) The same is true for the first series in Example 9.2.2, but the second series converges.
    🔗

    Example 9.2.4. Determining Convergence of a Series.

    Does the series \(\sum_{n=1}^{\infty}3^{-n}\) converge?
    🔗
    Solution.
    We’ll look at the sequence of partial sums. We have
    \begin{align*} S_{1} \amp = \frac{1}{3} \\ S_{2} \amp = \frac{4}{9} \\ S_{3} \amp = \frac{13}{27} \end{align*}
    and so on. It looks like the sequence of partial sums approaches \(\frac{1}{2}\text{,}\) so we guess that the series equals the same.
    🔗
    🔗
    🔗
    🔗

    Subsection Geometric Series

    The series in Example 9.2.4, as well as the first two series in Example 9.2.2, are examples of an important series known as a geometric series.
    🔗

    Definition 9.2.5. Geometric Series.

    A series \(\sum_{k=0}^{\infty}a_{k}\) is a geometric series if \(a_{k} = ar^{k}\) for some constants \(a\) and \(r\text{.}\) Equivalently, the terms of the series form a geometric sequence (see Example 9.1.7).
    🔗
    🔗
    Geometric series are useful because it’s straightforward to find their values. To see how, let \(\sum_{k=0}^{\infty}ar^{k}\) be a geometric series and let \(\{S_{n}\}\) denote the corresponding sequence of partial sums. Then
    \begin{equation*} S_{n} = a + ar + ar^{2} + \cdots + ar^{n} \end{equation*}
    which gives
    \begin{equation*} S_{n} - rS_{n} = a - ar^{n+1}\text{.} \end{equation*}
    We can solve this for \(S_{n}\) to get
    \begin{equation*} S_{n} = \frac{a - ar^{n+1}}{1 - r}\text{.} \end{equation*}
    🔗
    At this point, we can find the limit of the partial sums using Example 9.1.7. Therefore \(\sum_{k=0}^{\infty}ar^{k}\) converges to \(\frac{a}{1 - r}\) if \(|r| \lt 1\) and diverges otherwise.
    🔗
    As a quick example of this result, we can find the value of \(\sum_{n=1}^{\infty}3^{-n}\) since this series is geometric. To do so, we must determine \(a\) and \(r\) for this sum. Since
    \begin{equation*} \sum_{n=1}^{\infty}3^{-n} = \frac{1}{3} + \frac{1}{9} + \frac{1}{27} + \cdots\text{,} \end{equation*}
    we have \(a = \frac{1}{3}\) and \(r = \frac{1}{3}\) also. Hence the series sums to \(\frac{1/3}{2/3} = \frac{1}{2}\text{.}\)
    🔗

    Example 9.2.6. Computing a geometric series.

    Determine the value of \(\sum_{k=0}^{\infty}\left(-\frac{1}{4}\right)^{k} 5^{6 - k}\) if it exists.
    🔗
    Solution.
    Since this series contains terms being raised to the \(k^{\text{th}}\) power, we suspect it may be geometric. If we write out the first several terms, we get
    \begin{equation*} \sum_{k=0}^{\infty}\left(-\frac{1}{4}\right)^{k} 5^{6 - k} = 5^{6} - \frac{5^{5}}{4} + \frac{5^{4}}{4^{2}} - \cdots\text{,} \end{equation*}
    so at each step we’re dividing by \(4\cdot5 = -20\text{.}\) This series is therefore a geometric series with \(a = 5^{6}\) and \(r = -\frac{1}{20}\text{.}\) Since \(|r| \lt 1\text{,}\) this series converges. The value of this series is
    \begin{equation*} \sum_{k=0}^{\infty}\left(-\frac{1}{4}\right)^{k} 5^{6 - k} = \frac{a}{1 - r} = \frac{5^{6}}{21/20}\text{.} \end{equation*}
    🔗
    🔗
    🔗
    We can also find \(a\) and \(r\) without writing out the first few terms of the series.
    🔗

    Example 9.2.7. A divergent geometric series.

    Determine the value of \(\sum_{k=2}^{\infty}3^{4 - k}7^{3k}\text{.}\)
    🔗
    Solution.
    We can rewrite the series as
    \begin{equation*} \sum_{k=2}^{\infty}3^{4 - k}7^{3k} = \sum_{k=2}^{\infty}3^{4}\left(\frac{7^{3}}{3}\right)^{k}\text{.} \end{equation*}
    This is a geometric series with
    \begin{equation*} a = 3^{4}\frac{7^{6}}{3^{2}}\quad\text{and}\quad r = \frac{7^{3}}{3}\text{.} \end{equation*}
    Since \(|r| > 1\text{,}\) the series diverges.
    🔗
    🔗
    🔗

    Example 9.2.8. Evaluating \(.999\ldots\).

    Prove that \(.999\ldots = 1\) using geometric series.
    🔗
    Solution.
    First, we need to write \(.999\ldots\) as a geometric series. We can do so as follows:
    \begin{equation*} .999\ldots = \frac{9}{10} + \frac{9}{100} + \cdots\text{,} \end{equation*}
    and so we see that
    \begin{equation*} .999\ldots = \sum_{k=1}^{\infty}\frac{9}{10^{k}}\text{.} \end{equation*}
    This is a geometric series with \(a = \frac{9}{10}\) and \(r = \frac{1}{10}\) (and so is convergent!), and so
    \begin{equation*} \sum_{k=1}^{\infty}\frac{9}{10^{k}} = \frac{9/10}{1 - \frac{1}{10}} = 1\text{.} \end{equation*}
    🔗
    🔗
    🔗

    Example 9.2.9. Writing a decimal as a fraction.

    Rewrite the decimal \(.123451234512345\ldots\) as a fraction \(\frac{m}{n}\text{.}\)
    🔗
    Solution.
    First, it’s a mathematical fact that any repeating decimal can be written as a rational number so we know that we can actually write \(0.\overline{12345}\) as a fraction. We’ll do so by rewriting the decimal as a geometric series:
    \begin{equation*} 0.\overline{12345} = \frac{12345}{100000} + \frac{12345}{10000000000} + \cdots\text{,} \end{equation*}
    which is a geometric series with \(a = \frac{12345}{100000}\) and \(r = \frac{1}{100000}\text{.}\) This series is also convergent, and has sum
    \begin{equation*} \frac{a}{1 - r} = \frac{12345/100000}{99999/100000} = \frac{12345}{99999}\text{.} \end{equation*}
    🔗
    🔗
    🔗
    🔗

    Subsection Telescoping Series

    We can find exact values for geometric series since their partial sums simplify nicely. Another type of series that can be calculated (relatively) easily is the telescoping series, and for the same reason.
    🔗

    Definition 9.2.10. Telescoping Series.

    A telescoping series is a series that can be written in the form
    \begin{equation*} \sum[a_{k+1}-a_{k}]\text{.} \end{equation*}
    🔗
    🔗
    The partial sums of a telescoping series simplify quite a bit due to cancellation of successive terms. In particular, the partial sums look like
    \begin{equation*} \sum_{k=0}^n[a_{k+1}-a_{k}] = a_{n+1}-a_{0}\text{.} \end{equation*}
    Hence the value of the series can be found by computing
    \begin{equation*} \lim_{n\to\infty}[a_{n+1}-a_{0}]\text{.} \end{equation*}
    We’ll demonstrate by way of example.
    🔗

    Example 9.2.11. Telescoping logarithms.

    Find \(\sum_{k=1}^{\infty}\ln\parens{\frac{k + 1}{k}}\text{.}\)
    🔗
    Solution.
    If we write out the first few terms, we get
    \begin{equation*} \sum_{k=1}^{\infty}\ln\parens{\frac{k + 1}{k}} = (\ln2 - \ln 1) + (\ln3 - \ln2) + \cdots \end{equation*}
    so it looks like many of these terms cancel each other out. To be precise about this, we’ll find the partial sums of this series and then consider their limit:
    \begin{equation*} S_{n} = \sum_{k=1}^{n}\ln\parens{\frac{k+1}{k}} = \ln(n + 1)\text{,} \end{equation*}
    which goes to \(\infty\) as \(n\to\infty\text{.}\) So the series diverges.
    🔗
    🔗
    🔗
    Not every series is quickly recognizable as a telescoping series. Occasionally, we need to do a bit of algebra first.
    🔗

    Example 9.2.12. Recognizing a telescoping series.

    Find \(\sum_{n=3}^{\infty}\frac{3}{n(n-1)}\text{.}\)
    🔗
    Solution.
    It’s not obvious at all that the series is telescoping, even if we write out a few terms. However, if we try partial fractions on \(\frac{3}{n(n-1)}\) we obtain (see SageMath cell below)
    \begin{equation*} \frac{3}{n(n-1)} = \frac{3}{n - 1} - \frac{3}{n}\text{.} \end{equation*}
    So
    \begin{equation*} \sum_{n=3}^{\infty}\frac{3}{n(n-1)} = \sum_{n=3}^{\infty}\left[\frac{3}{n-1} - \frac{3}{n}\right]\text{.} \end{equation*}
    🔗
    The \(k^{\text{th}}\) partial sum is
    \begin{equation*} S_{k} = \sum_{n=3}^{k}\left[\frac{3}{n-1} - \frac{3}{n}\right] = \frac{3}{2} - \frac{3}{k}\text{,} \end{equation*}
    and so \(S_{k}\to \frac{3}{2}\text{.}\) Hence \(\sum_{n=3}^{\infty}\frac{3}{n(n-1)} = \frac{3}{2}\text{.}\)
    🔗
    🔗
    🔗
    🔗
    Series, or rather the summation symbol \(\Sigma\text{,}\) obey many of the same laws as integrals: they split over sums and we may pull constants out.
    🔗
