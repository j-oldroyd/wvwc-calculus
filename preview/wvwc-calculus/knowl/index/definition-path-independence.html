Definition 14.3.2. Path independence.

A vector field \(\vb{F}\) is path independent on a domain \(D\) if
\begin{equation*} \int_{C_{1}}\vb{F}\dr = \int_{C_{2}}\vb{F}\cdot d\vb{r} \end{equation*}
for any piecewise smooth paths in \(D\) with the same endpoints. Equivalently, \(\vb{F}\) is path independent if
\begin{equation*} \oint_{C}\vb{F}\cdot d\vb{r} = 0 \end{equation*}
for any piecewise smooth closed curve \(C\) in \(D\text{.}\)
