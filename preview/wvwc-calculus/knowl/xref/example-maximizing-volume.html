Example 12.8.4. Maximizing volume.

A lidless box may be made from 12 m2 of material. What dimensions give the maximum volume?
🔗
Solution.
If we denote the dimensions of the box by \(x,y,z\text{,}\) then the function we need to maximize is \(V(x,y,z) = xyz\text{.}\) However, not all dimensions are valid since we’re only allowed to use 12 m2 of material. This means that the surface area of the box, which is given by \(A(x,y,z) = xy + 2xz + 2yz\text{,}\) must be equal to \(12\text{.}\) So our constraint is that \(A(x,y,z) = 12\text{,}\) and because we have this constraint we’ll try to solve this using Algorithm 12.8.3.
🔗
To begin, we need to solve \(\grad V = \lambda \grad A\text{.}\) Equivalently, we need to solve the system
\begin{align*} V_{x} & = \lambda A_{x} \\ V_{y} & = \lambda A_{y} \\ V_{z} & = \lambda A_{z} \end{align*}
which turns into
\begin{align*} yz & = \lambda(y + 2z) \\ xz & = \lambda(x + 2z) \\ xy & = \lambda(2x + 2y) \end{align*}
Now, we don’t actually need to find a value for \(\lambda\text{.}\) We just need values for \(x,y,z\text{.}\) If we multiply these equations by \(x\text{,}\) \(y\) and \(z\) respectively, then we get
\begin{align*} xyz & = \lambda(xy + 2xz) \\ xyz & = \lambda(xy + 2yz) \\ xyz & = \lambda(2xz + 2yz) \end{align*}
or just
\begin{equation*} \lambda(xy + 2xz) = \lambda(xy + 2yz) = \lambda(2xz + 2yz) \end{equation*}
We can also divide by \(\lambda\text{,}\) since there’s no way \(\lambda\) could equal \(0\) and satisfy the above equations (since \(x,y,z\) are all positive). This gives us
\begin{equation} xy + 2xz = xy + 2yz = 2xz + 2yz\tag{12.8.1} \end{equation}
🔗
The first two expressions in (12.8.1) simplify to \(x=y\text{,}\) while the first and third reduce to \(x = 2z\text{.}\) So
\begin{equation*} x = y = 2z\text{.} \end{equation*}
But these variables also need to satisfy \(A(x,y,z) = 12\text{:}\)
\begin{align*} 12 & = xy + 2xz + 2yz \\ & = 4z^{2} + 4z^{2} + 4z^{2} \\ & = 12z^{2} \end{align*}
Therefore \(z = \pm1\text{,}\) so we take \(z=1\) (since we’re dealing with dimensions). This also means that \(x = y = 2(1) = 2\text{.}\) So the point \((2,2,1)\) is therefore our candidate for the dimensions that maximize volume subject to the constraint \(A = 12\text{.}\)
🔗
In order to actually verify that \(V(2,2,1) = 4\) is actually the largest possible volume we can obtain, we need to show that it’s actually a maximum and not a minimum. This means we need to find another point \((x,y,z)\) that also satisfies \(A = 12\text{,}\) and show that it gives a smaller value for \(V\text{.}\) One such point is just \((-2,-2,-1)\text{,}\) and \(V(-2,-2,-1) = -4 < V(2,2,1)\text{.}\) So \(4\) is actually a maximum value and not a minimum value.
