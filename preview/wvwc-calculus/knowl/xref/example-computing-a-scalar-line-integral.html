Example 14.2.2. Computing a scalar line integral.

Compute \(\int_{C}(xy)^{1/3}\dd{s}\text{,}\) where \(C\) is the segment of the parabola \(y = x^{2}\) from \(x = 0\) to \(x = 1\text{.}\)
🔗
Solution.
The curve \(C\) is traced out by the vector function \(\vb{r}(t) = \dotprod{t,t^{2}}\) for \(0\leq t\leq 1\text{.}\) Therefore
\begin{align*} \int_{C}(xy)^{1/3}\dd{s} \amp = \int_{0}^{1} t\sqrt{1 + 4t^{2}}\,dt\\ \amp = \frac{1}{8}\int_{1}^{5}\sqrt{u}\,du\\ \amp = \frac{5^{3/2} - 1}{12} \end{align*}
