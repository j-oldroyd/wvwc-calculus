Example 9.7.2. Taylor polynomial of \(e^x\).

Compute \(T_3(x)\) for \(f(x)=e^x\) centered at \(0\text{.}\)

Solution.
Using DefinitionÂ 9.7.1, we need to find
\begin{equation*} T_3(x) = \sum_{k=0}^3\frac{f^{(k)}(0)}{k!}x^k\text{.} \end{equation*}
This is straightforward to find since \(f^{(k)}(0) = 1\) for all \(k\text{,}\) and so
\begin{equation*} T_3(x) = 1 + x + \frac{x^2}{2} + \frac{x^3}{6}\text{.} \end{equation*}
