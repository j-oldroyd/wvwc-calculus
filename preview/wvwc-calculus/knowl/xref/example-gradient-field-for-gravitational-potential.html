Example 14.1.8. Gradient Field for Gravitational Potential.

Find the gradient field associated to
\begin{equation*} \varphi(x,y,z) = \frac{mMG}{\sqrt{x^{2} + y^{2} + z^{2}}}\text{.} \end{equation*}
🔗
Solution.
We can find the gradient field right away:
\begin{equation*} \grad\varphi = \frac{-mMG}{(x^{2} + y^{2} + z^{2})^{3/2}}\dotprod{x,y,z}\text{.} \end{equation*}
