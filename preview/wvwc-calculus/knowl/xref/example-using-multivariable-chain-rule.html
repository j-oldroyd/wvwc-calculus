Example 12.5.3. Using the multivariable chain rule.

Let \(f(x,y,z) = \ln\sqrt{x^{2}+y^{2}+z^{2}}\) where
\begin{equation*} x = \sin t,\quad y = \cos t\quad\text{and } z = \tan t. \end{equation*}
Find \(\dv{f}{t}\bigg]_{t=0}\text{.}\)

Now, to find \(\dv{f}{t}\) we need to use the chain rule, which gives
\begin{align*} \dv{f}{t} & = f_{x}x'(t) + f_{y}y'(t) + f_{z}z^\prime(t) \\ & = \frac{x}{x^{2} + y^{2} + z^{2}}\cos t - \frac{y}{x^{2} + y^{2} + z^{2}}\sin t + \frac{z}{x^{2} + y^{2} + z^{2}}\sec^{2}t\\ & = \frac{\sin t\cos t}{\sec^{2}t} - \frac{\cos t\sin t}{\sec^{2}t} + \frac{\tan t}{\sec^{2}t}\\ & = \cos t\sin t \\ & = \frac{1}{2}\sin2t \end{align*}
So
\begin{equation*} \dv{f}{t}\bigg]_{t=0} = 0. \end{equation*}
