Example 13.5.5. Finding an average value.

Find the average value of the function \(f(x,y,z) = xyze^{-x^{2} - y^{2}}\) over the region \(D\) given in Example 13.5.4.
🔗
Solution.
First, let \(\operatorname{Vol}(D)\) denote the volume of \(D\text{.}\) Then the average value of \(f\) over \(D\) is just
\begin{equation*} \frac{\iiint_{D}xyze^{-x^{2} - y^{2}}\,dV}{\operatorname{Vol}(D)} = \frac{\frac{3}{64}}{\sqrt{\ln2\ln4}} \approx .05 \end{equation*}
