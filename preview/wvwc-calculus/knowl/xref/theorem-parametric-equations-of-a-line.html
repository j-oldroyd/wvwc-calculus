Theorem 11.5.4. Parametric equations of a line.

If \(P_{0}(x_{0},y_{0},z_{0})\) is a point on a line \(L\text{,}\) and if \(\mathbf{v} = \dotprod{a,b,c}\) is parallel to \(L\text{,}\) then
\begin{equation*} \mathbf{r} = \mathbf{r}_{0}+t\mathbf{v} \end{equation*}
traces out \(L\text{,}\) where
\begin{equation*} \mathbf{r} = \dotprod{x,y,z},\quad\mathbf{r}_{0} = \dotprod{x_{0},y_{0},z_{0}}. \end{equation*}
We can also write this as the system of equations
\begin{equation*} x = x_{0}+ta,\quad y=y_{0}+tb\quad\text{and}\quad z = z_{0}+tc. \end{equation*}
