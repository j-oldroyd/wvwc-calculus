Example 2.1.4. Limit of a piecewise function.

Let
\begin{equation*} f(x) = \begin{cases} \sin x & \text{if $x<0$} \\ \cos x & \text{otherwise} \end{cases}. \end{equation*}
Find \(\lim\limits_{x\to0}f(x)\text{.}\)
ðŸ”—
Solution.
If we graph this function, we see that at \(x=0\) there is a jump in the graph. In particular, if \(x\) approaches \(0\) from the left then \(f(x)\) approaches \(0\text{,}\) whereas if \(x\) approaches \(0\) from the right then \(f(x)\) approaches \(1\text{.}\) So this function does not appear to have an unambiguous limit as \(x\) approaches \(0\text{.}\) Another way to say this: \(\lim\limits_{x\to0}f(x)\) does not exist.
