Example 14.3.8. Finding a line integral.

Let \(\vb{F} = (3x^{2} + 2y^{2})\vb{i} + (4xy - e^{-y^{2}})\vb{j}\) and let \(C\) denote the ellipse
\begin{equation*} \frac{(x-1)^{2}}{9} + \frac{(3y + 2)^{2}}{10} = 5\text{.} \end{equation*}
Find \(\oint_{C}\vb{F}\cdot\vb{T}\dd{s}\text{.}\)
🔗
Solution.
We can parameterize \(C\) and then compute \(\oint_{C}\vb{F}\dr\) as in Section 14.2, but we’ll first check if \(\vb{F}\) is conservative. Since \(\pdv{Q}{x} = 4y = \pdv{P}{y}\text{,}\) it follows that \(\vb{F}\) is conservative. Hence \(\oint_{C}\vb{F}\dr = 0\text{.}\)
