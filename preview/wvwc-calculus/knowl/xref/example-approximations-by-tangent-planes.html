Example 12.4.2. Approximations by tangent planes.

Find the tangent plane to \(f(x,y) = xe^{xy}\) at the point \((2,0,2)\text{.}\) Use this to approximate \(f(1.9,0)\)

Solution.
The equation of the tangent plane is given by
\begin{equation*} z - 2 = (x - 2) + 4(y - 0) \end{equation*}
which we can rewrite as
\begin{equation*} z = x + 4y. \end{equation*}
So at \((1.9,0)\text{,}\) we should have
\begin{equation*} f(1.9,0)\approx 1.9 + 4(0) = 1.9. \end{equation*}
