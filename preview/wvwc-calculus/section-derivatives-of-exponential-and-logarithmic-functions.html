    Section 4.3 Derivatives of exponential and logarithmic functions

    Subsection Inverse function review

    Suppose we want to solve the equation \(y = f(x)\) for \(x\text{.}\) Then we can only do so unambiguously if \(f\) is an invertible function.
    🔗

    Definition 4.3.1. Invertible functions.

    A function \(f(x)\) is invertible if and only if it is a one-to-one function. That is, \(f(x_{1}) = f(x_{2})\) forces \(x_{1} = x_{2}\text{.}\)
    🔗
    🔗
    Since a function is one-to-one if and only if its graph passes the horizontal line test, the horizontal line test also gives us a useful way to check if a function is invertible.
    🔗

    Example 4.3.2.

    Let \(f(x) = e^{x}\text{,}\) the natural exponential function introduced in Section 4.1. Then \(f(x)\) is an invertible function, since its graph passes the horizontal line test.
    🔗
    🔗
    If a function \(f\) is invertible, then there exists an inverse function \(f^{1}\) satisfying the following equations:
    \begin{align*} f(f^{-1}(y)) & = y \\ f^{-1}(f(x)) & = x. \end{align*}
    Essentially, \(f^{-1}\) is the rule that undoes the transformation that \(f\) applies to \(x\text{.}\) The domain of \(f^{-1}\) is the range of \(f\) and the range of \(f^{-1}\) is the domain of \(f\text{.}\) Note that \(f^{-1}\neq\frac{1}{f}\text{!.}\)
    🔗
    The graph of an inverse function is related to the graph of the original function in a very nice way. To get the graph of \(f^{-1}\) from \(f\text{,}\) just reflect the graph of \(f\) across the line \(y=x\text{.}\)
    🔗

    Example 4.3.3. Graphing the inverse of the natural exponential.

    Let \(f(x) = e^{x}\text{.}\) Graph \(f^{-1}(x)\text{.}\)
    🔗
    Solution.
    If we graph \(e^{x}\) and then rotate the graph across the line \(y=x\text{,}\) we get the graph in Figure 4.3.4.
    🔗
    🔗
    🔗
    🔗

    Subsection Calculus with inverse functions

    There are two important things to notice about the graph of the inverse function in Example 4.3.2: it’s continuous and differentiable. This is because the same was true of the original graph, and if we think about it this should be true in general as well since graphing the inverse doesn’t add any new gaps or cusps to the graph.
    🔗

    Theorem 4.3.5. Continuity and differentiability of inverses.

    Let \(f(x)\) be a continuous (respectively, differentiable) function. Suppose that \(f(x)\) is invertible, and has inverse \(f^{-1}(x)\text{.}\) Then \(f^{-1}(x)\) is continuous (respectively, differentiable).
    🔗
    🔗

    Example 4.3.6. Finding the derivative of an inverse function.

    Let \(f(x) = \sqrt{x-1}\) and note that \(f\) is invertible. Find the derivative of \(f^{-1}\text{.}\)
    🔗
    Solution.
    One way to do this is just start by finding \(f^{-1}\text{.}\) So we’ll set \(y = \sqrt{x-1}\text{,}\) solve for \(x\) and then switch \(x\) and \(y\text{.}\) If we do this, we get
    \begin{equation*} y = x^{2} + 1 \end{equation*}
    so \(f^{-1}(x) = x^{2} + 1\) (and note that the domain of \(f^{-1}\) is \([0,\infty)\) since this is the range of \(f\text{!}\)). Hence
    \begin{equation*} \dv{}{x}f^{-1}(x) = 2x\text{.} \end{equation*}
    🔗
    🔗
    🔗
    The method used in Example 4.3.6 will certainly work in simple cases. But what do we do if we can’t (or don’t want to) find an explicit formula for the inverse function? The following formula will help us to do this.
    🔗

    Theorem 4.3.7. Derivative of an inverse function.

    Let \(f(x)\) be a differentiable function with inverse \(f^{-1}(x)\text{.}\) Assume that \(f^{-1}\) is itself differentiable. Then
    \begin{equation*} (f^{-1})'(x) = \frac{1}{f'(f^{-1}(x))}\text{.} \end{equation*}
    🔗
    🔗

    Proof.

    First, let \(y = f^{-1}(x)\text{,}\) so that we need to find \(y'\text{.}\) Then we can say that \(f(y) = x\text{.}\) Now differentiate this equation implicitly to get
    \begin{equation*} f'(y)y' = 1 \end{equation*}
    or just
    \begin{equation*} y' = \frac{1}{f'(y)} = \frac{1}{f'(f^{-1}(x))}\text{.} \end{equation*}
    🔗
    🔗

    Example 4.3.8. Exponential inverse revisited.

    Using the fact that the derivative of \(e^{x}\) is itself, find the derivative of its inverse.
    🔗
    Solution.
    Let \(f(x) = e^{x}\text{.}\) Then by Theorem 4.3.7 we have
    \begin{equation*} (f^{-1})'(x) = \frac{1}{f'(f^{-1}(x))} = \frac{1}{f(f^{-1}(x))} = \frac{1}{x}\text{.} \end{equation*}
    So the derivative of the inverse of \(e^{x}\) is \(\frac{1}{x}\text{.}\)
    🔗
    🔗
    🔗

    Example 4.3.9. Derivative at a point.

    Let \(f(x) = x^{3} + 3\sin x + 2\cos x\text{.}\) Find \((f^{-1})'(2)\)
    🔗
    Solution.
    By Theorem 4.3.7, we know that
    \begin{equation*} (f^{-1})'(2) = \frac{1}{f'(f^{-1}(2))}\text{.} \end{equation*}
    By inspection, \(f(0) = 2\) which means \(f^{-1}(2) = 0\text{.}\) Since
    \begin{equation*} f'(x) = 3x^{2} + 3\cos x - 2\sin x\text{,} \end{equation*}
    this gives
    \begin{equation*} (f^{-1})'(2) = \frac{1}{f'(f^{-1}(2))} = \frac{1}{f'(0)} = \frac{1}{3}\text{.} \end{equation*}
    🔗
    🔗
    🔗
    🔗

    Subsection Derivatives of exponentials

    We’ve already mentioned that \(\dv{}{x}e^{x} = e^{x}\text{,}\) but we’ve said nothing about differentiating functions such as \(3^{x}\) or \((\frac{1}{\pi})^{-x}\text{.}\) However, it turns out that we can get these derivatives from the derivative of \(e^{x}\) without too much trouble.
    🔗

    Theorem 4.3.10. Derivatives of exponential functions.

    Let \(a \gt 0\) and set \(f(x) = a^{x}\text{.}\) Then \(f'(x) = a^{x}\ln a\text{.}\)
    🔗
    🔗

    Proof.

    To prove this, we need to use what’s available to us: the derivative of \(e^{x}\text{.}\) From Example 4.2.2 we say that \(e^{x\ln 3} = 3^{x}\text{.}\) More generally, we can also say that \(e^{x\ln a} = a^{x}\text{.}\) Hence
    \begin{align*} \dv{}{x}a^{x} & = \dv{}{x}e^{x\ln a} \\ & = (\ln a)e^{x\ln a} \\ & = (\ln a)a^{x} \text{.} \end{align*}
    🔗
    🔗

    Example 4.3.11. Differentiating an exponential.

    Let \(g(t) = 2.6^{3t - \sin t}\text{.}\) Find \(g'(t)\text{.}\)
    🔗
    Solution.
    If we set \(u = 3t - \sin t\text{,}\) then \(g(t) = g(u) = 2.6^{u}\text{.}\) By the chain rule, \(g'(t) = g'(u)\dv{u}{t} = (\ln2.6)2.6^{u}(3-\cos t) = (\ln2.6)2.6^{3t-\sin t}(3-\cos t)\text{.}\)
    🔗
    🔗
    🔗
    Although we certainly want to know the derivative of \(a^{x}\text{,}\) or at least how to find it, most applications involving exponential functions use the natural exponential function \(e^{x}\) instead. The derivative of \(e^{x}\) is probably the most important derivative in this course.
    🔗

    Example 4.3.12. Solutions of a differential equation.

    Show that \(x(t) = Ae^{-t} + Bte^{-t}\) satisfies the differential equation \(x'' + 2x' + x = 0\text{,}\) where \(A,B\) are arbitrary constants.
    🔗
    Solution.
    We need to show that if we compute \(x'\) and \(x''\) and plug these expressions into the differential equation, this will simplify out to \(0\text{.}\) Since
    \begin{align*} x' & = -Ae^{-t} +Be^{-t} - Bte^{-t} \\ x'' & = Ae^{-t} - Be^{-t} - (Be^{-t} - Bte^{-t}) \\ & = Ae^{-t} - 2Be^{-t} + Bte^{-t} \text{,} \end{align*}
    it follows that
    \begin{align*} x'' + 2x' + x & = (Ae^{-t} - 2Be^{-t} + Bte^{-t}) + 2(-Ae^{-t} + Be^{-t} - Bte^{-r}) + Ae^{-t} + Bte^{-t} \\ & = 0 \text{.} \end{align*}
    🔗
    🔗
    🔗
    Exponential functions often appear in the solutions of differential equations, which are themselves often viewed as mathematical models of physical systems. Hence exponential functions play a significant role in predicting physical quantities, which goes a long way towards justifying their importance.
    🔗
    🔗

    Subsection Derivatives of logarithms

    Just as we can get the derivative of every exponential function \(a^{x}\) just by knowing the derivative of \(e^{x}\text{,}\) we can get the derivative of every logarithmic function \(\log_{a}x\) just by knowing that \(\dv{}{x}\ln x = \frac{1}{x}\text{.}\)
    🔗

    Theorem 4.3.13. Derivatives of logarithmic functions.

    Let \(a > 0, a\neq 1\) and set \(f(x) = \log_{a}x\text{.}\) Then
    \begin{equation*} f'(x) = \frac{1}{x\ln a}\text{.} \end{equation*}
    🔗
    🔗

    Example 4.3.14. Differentiating nested logarithms.

    Let \(f(x) = \log_{2}(\log_{5} x)\text{.}\) Find \(f'(x)\text{.}\)
    🔗
    Solution.
    By the chain rule, we have
    \begin{equation*} f'(x) = \frac{1}{(\log_{5}x)\ln 2}\dv{}{x}\log_{5}x = \frac{1}{(\log_{5} x)\ln 2}\frac{1}{x\ln 5}\text{,} \end{equation*}
    🔗
    🔗
    🔗
    Logarithms can be used to greatly simplify derivatives involving products, division or exponentiation using a technique known as logarithmic differentiation.
    🔗

    Example 4.3.15. A simple fraction.

    Let \(f(x) = \frac{e^{-x}\cos x}{x^{2} + 4x + 4}\text{.}\) Find \(f'(x)\text{.}\)
    🔗
    Solution.
    We can find \(f'(x)\) without resorting to logarithms, but this would require using the product, quotient and chain rules. The algebra would be awful. So we’ll using logarithms instead! Set \(y = f(x)\text{.}\) Then
    \begin{equation*} \ln y = \ln e^{-x} + \ln\cos x - \ln (x^{2} + 4x + 4) = -x + \ln\cos x - 2\ln(x+2)\text{.} \end{equation*}
    Now we differentiate both sides implicitly to obtain
    \begin{equation*} \frac{y'}{y} = -1 - \frac{\sin x}{\cos x} - \frac{2}{x+2}\text{.} \end{equation*}
    🔗
    Hence
    \begin{equation*} y' = y\left(-1 - \tan x - \frac{2}{x+2}\right) = \frac{e^{-x}\cos x}{x^{2} + 4x + 4}\left(-1 - \tan x - \frac{2}{x+2}\right)\text{.} \end{equation*}
    🔗
    🔗
    🔗
    Logarithmic differentiation is useful for finding derivatives of expressions containing complicated quotients, products or powers.
    🔗

    Example 4.3.16. A simple exponent.

    Let \(h(z) = z^{z}\text{.}\) Find \(h'(z)\text{.}\)
    🔗
    Solution.
    We’ll use logarithmic differentiation again to simplify \(h\) and remove the exponent. Set \(y = h(z)\text{,}\) which gives
    \begin{equation*} \ln y = z\ln z\text{.} \end{equation*}
    So
    \begin{equation*} \frac{y'}{y} = \ln z + 1\text{,} \end{equation*}
    which means that
    \begin{equation*} y' = y(\ln z + 1) = z^{z}(\ln z + 1)\text{.} \end{equation*}
    🔗
