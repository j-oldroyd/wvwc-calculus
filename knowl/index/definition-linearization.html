Definition 12.4.3. Linearization.

Let \(f(x,y)\) be a function for which \(f_{x}\) and \(f_{y}\) are both continuous at a point \((x_{0},y_{0})\text{.}\) Then the linearization of \(f(x,y)\) at \((x_{0},y_{0})\) is the function \(L(x,y)\) given by
\begin{equation*} L(x,y) = f(x_{0},y_{0}) + f_{x}(x_{0},y_{0})(x-x_{0}) + f_{y}(x_{0},y_{0})(y-y_{0}). \end{equation*}
