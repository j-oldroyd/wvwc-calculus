Theorem 11.7.8. Vector Derivative Rules.

Let \(\mathbf{u}\) and \(\mathbf{v}\) be differentiable vector functions, \(c\) be a scalar and let \(f(t)\) be a differentiable (scalar) function. Then the following formulas hold:
  1. \(\displaystyle \dv{}{t}[\mathbf{u}(t)+\mathbf{v}(t)] = \mathbf{u}'(t)+\mathbf{v}'(t)\)
  2. \(\displaystyle \dv{}{t}[c\mathbf{u}(t)] = c\mathbf{u}^{\prime}(t)\)
  3. \(\displaystyle \dv{}{t}[f(t)\mathbf{u}(t)] = f^{\prime}(t)\mathbf{u}(t)+f(t)\mathbf{u}^{\prime}(t)\)
  4. \(\displaystyle \dv{}{t}[\mathbf{u}(t)\cdot\mathbf{v}(t)] = \mathbf{u}^{\prime}(t)\cdot\mathbf{v}(t)+\mathbf{u}(t)\cdot\mathbf{v}^{\prime}(t)\)
  5. \(\displaystyle \dv{}{t}[\mathbf{u}(t)\times\mathbf{v}(t)] = \mathbf{u}^{\prime}(t)\times\mathbf{v}(t)+\mathbf{u}(t)\times\mathbf{v}^{\prime}(t)\)
  6. \(\displaystyle \dv{}{t}\mathbf{u}(f(t)) = f^{\prime}(t)\mathbf{u}^{\prime}(f(t))\)
