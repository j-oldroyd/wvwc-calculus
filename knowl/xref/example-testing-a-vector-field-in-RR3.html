Example 14.3.9. Testing a vector field in \(\RR^{3}\).

Let \(\vb{F} = \dotprod{e^{x}\cos y + yz, xz - e^{x}\sin y, xy + z}\text{.}\) Determine if \(\vb{F}\) is conservative.
This is taken from Example 2 on page 1165 of Thomasâ€™ Calculus, \(11^{\text{th}}\) edition.
Solution.
We need to check if \(\curl\vb{F} = \vb{0}\text{.}\) So we compute this like a cross product, giving \(\vb{0}\text{.}\) Hence the vector field is irrotational and therefore conservative.
